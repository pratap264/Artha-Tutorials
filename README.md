# Artha Tutorials - PUC Coaching Center Website

A modern, responsive website for **Artha Tutorials**, a premier PUC (11th & 12th Board) coaching center.

## Features

- **Multiple Streams**: Science, Commerce, and Humanities course offerings
- **Comprehensive Curriculum**: Detailed subject information and course structure
- **Student Portal**: Course information, admissions details, and results
- **Contact & Admissions**: Easy enrollment and communication channels
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Built with React, Vite, and Tailwind CSS

## Available Streams

### Science Stream
- Physics, Chemistry, Biology, Mathematics

### Commerce Stream  
- Accountancy, Economics, Business Studies, English

### Humanities Stream
- History, Political Science, Geography, English

## Technologies Used

- React 18.3
- Vite
- Tailwind CSS
- React Router DOM
- React Icons
- AOS (Animate On Scroll)

## Getting Started

```bash
npm install
npm run dev
```
